# side_car

0. 메인 로그인 화면
  - 사이드카가 거리를 달리는 애니메이션, 하단에 로그인 영역

1. 회원가입
  - 처음 시작을 회원가입을 하는 걸로 시작

2. 메인 게임 기능
  - 게임 설명
  - 지도 화면(주제 선택)
  - 게임 실행

3. 회원 정보
  - 레벨 표시
  - 거리 기록
  - 등수 표기

4. leaderboard
  - 나라, 아이디, 거리 표시
  - 거리에 따른 등수 표시
